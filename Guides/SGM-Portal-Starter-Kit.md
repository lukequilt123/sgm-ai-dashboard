# SGM Client Portal — Starter Kit

Use this document to kick off a new portal project that follows the same design language as the existing SGM Client Portal. Hand this file to any developer (or AI assistant) and they should be able to produce a pixel-consistent result.

---

## Project Overview

You are building a **client-facing portal** for Sleeping Giant Media (SGM). It is a static site hosted on **GitHub Pages** that pulls its data from a **Google Sheet** via **Google Apps Script** pushing JSON to the repo. The portal is password-protected with a simple SHA-256 hash check on the client side.

### Architecture

```
Google Sheet (source of truth)
    │
    ▼
Google Apps Script (reads sheet, builds JSON, pushes via GitHub API)
    │
    ▼
GitHub Repo (static site + data/client-data.json)
    │
    ▼
GitHub Pages (serves the portal)
    │
    ▼
Browser (password gate → dashboard)
```

### Tech Stack
- **Frontend:** Vanilla HTML, CSS, JavaScript (no frameworks, no build tools)
- **Data pipeline:** Google Apps Script → GitHub REST API
- **Hosting:** GitHub Pages
- **Auth:** Client-side SHA-256 password hash (stored in `js/config.js`)

## Step-by-Step Setup

### 1. Create the GitHub Repo
- Create a new repo (e.g., `your-org/project-name`)
- Enable GitHub Pages (Settings → Pages → Source: `main` branch, `/ (root)`)
- Generate a **Fine-grained Personal Access Token** with Contents read/write scope for this repo

### 2. Create the Google Sheet
- Set up your data sheet with headers in a header row (e.g., row 6)
- Data rows start from the row below the headers
- Include a config area (e.g., cells for client name, account manager, etc.)

### 3. Set Up Google Apps Script
- Open the sheet → Extensions → Apps Script
- Add Script Properties (Project Settings → Script Properties):
  - `GITHUB_TOKEN` = your GitHub PAT
  - `GITHUB_REPO` = `owner/repo-name`
  - `GITHUB_PATH` = `data/client-data.json`
- Paste the Apps Script code (template below)
- Create a custom menu trigger via `onOpen()` so users can push data with one click

### 4. Build the Frontend
- Use the design system documented below
- Load data via `fetch('data/client-data.json')` — no API keys needed, it's a static file
- Implement the password gate (template below)

## Design System

### Philosophy
**Dark glass-morphism with gradient accents.** Surfaces are semi-transparent with `backdrop-filter: blur()`. Brand colour comes from a pink-to-purple gradient used sparingly on buttons, badges, and accent lines. Everything else is monochrome white-on-dark.

### Brand Assets
- **Logo (white, for dark backgrounds):** `https://cdn.brandfetch.io/id6AGii4MS/w/396/h/126/theme/dark/logo.png`
- **Primary colours:** `#26153A` (dark purple), `#7F5FCE` (purple), `#FF1CC0` (pink)
- **Fonts:** Oswald (display/headings) + Open Sans (body/UI)

**Staggered fade-in** — add `fade-in` class to grid items:

**Hover states** (always present on interactive elements):
- Cards: `translateY(-3px)` over 0.3s
- Buttons: `translateY(-1px)` over 0.2s
- Small elements: `scale(1.05)` over 0.2s

### Responsive Breakpoints

| Breakpoint | Changes |
|---|---|
| `> 1024px` | Full grid layouts |
| `768–1024px` | 2-column grids, reduced padding |
| `< 768px` | Single column, hamburger nav, tables become stacked cards |
| `< 480px` | Everything single column |

**Mobile table pattern:** Hide `<thead>`, make each `<tr>` a card (`display: block`), show labels via `td::before { content: attr(data-label) }`. Always add `data-label` attributes to `<td>` elements.

### Design Rules

**DO:**
- Use CSS custom properties for all colours, radii, shadows
- Keep Oswald ALWAYS uppercase with letter-spacing
- Use the gradient sparingly — buttons, accent bars, badges only
- Add `fade-in` to grid items for page-load animation
- Include hover/focus states on every interactive element
- Use the purple focus ring on all focusable elements

**DON'T:**
- Use solid mid-tone backgrounds — surfaces are always semi-transparent
- Use Oswald in sentence case
- Use the gradient as a large background fill
- Use pure black — darkest is `--bg-deepest` (#0e0918)
- Use sharp corners — minimum 8px radius
- Skip hover states on interactive elements
- Hardcode colours — always reference tokens

### Password Gate (js/auth.js)

```javascript
const Auth = {
  isAuthenticated() {
    if (sessionStorage.getItem('sgm_auth') === 'true') {
      return true;
    }
    this.showGate();
    return false;
  },

  showGate() {
    document.querySelector('.portal-page').classList.add('hidden');

    const gate = document.createElement('div');
    gate.className = 'password-gate';
    gate.innerHTML = `
      <div class="password-card">
        <img src="https://cdn.brandfetch.io/id6AGii4MS/w/396/h/126/theme/dark/logo.png"
             alt="SGM" class="password-card__logo">
        <h1 class="password-card__title">Client Portal</h1>
        <p class="password-card__subtitle">Enter your access code to continue</p>
        <input type="password" class="password-card__input" id="pw-input"
               placeholder="Access code" autocomplete="off">
        <button class="password-card__submit" id="pw-submit">Enter Portal</button>
        <p class="password-card__error hidden" id="pw-error">Incorrect access code</p>
      </div>`;
    document.body.appendChild(gate);

    const submit = () => {
      const input = document.getElementById('pw-input');
      this.hashPassword(input.value).then(hash => {
        if (hash === CONFIG.passwordHash) {
          sessionStorage.setItem('sgm_auth', 'true');
          gate.remove();
          document.querySelector('.portal-page').classList.remove('hidden');
          // Trigger page init here if needed
          location.reload();
        } else {
          document.getElementById('pw-error').classList.remove('hidden');
          input.value = '';
          input.focus();
        }
      });
    };

    document.getElementById('pw-submit').addEventListener('click', submit);
    document.getElementById('pw-input').addEventListener('keydown', (e) => {
      if (e.key === 'Enter') submit();
    });

    setTimeout(() => document.getElementById('pw-input').focus(), 100);
  },

  async hashPassword(password) {
    const encoder = new TextEncoder();
    const data = encoder.encode(password);
    const hashBuffer = await crypto.subtle.digest('SHA-256', data);
    return Array.from(new Uint8Array(hashBuffer))
      .map(b => b.toString(16).padStart(2, '0'))
      .join('');
  }
};
```

```javascript
function pushToPortal() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName("Sheet1");  // ← Change to your sheet name
  const ui = SpreadsheetApp.getUi();

  if (!sheet) {

    return;
  }

  // ── Read config cells (adjust these to match your sheet layout) ──
  // Example: const clientName = sheet.getRange("B1").getValue();

  const headerRow = 1;        // ← Row number containing your headers
  const dataStartRow = 2;     // ← First row of data
  const lastRow = sheet.getLastRow();
  const lastCol = sheet.getLastColumn();

  if (lastRow < dataStartRow) {

    return;
  }

  const dataRows = sheet.getRange(dataStartRow, 1, lastRow - dataStartRow + 1, lastCol).getValues();

  // ── Build JSON ──
  const items = [];
  for (let i = 0; i < dataRows.length; i++) {
    const row = dataRows[i];

    // Skip empty rows
    if (!row[0] && !row[1]) continue;

    // Build your object — map columns to field names
    // Example:

    //   status: row[1] ? row[1].toString().trim() : "",
    //   dueDate: formatDateForJson(row[2]),
    //   description: row[3] ? row[3].toString().trim() : "",
    // });
  }

    ui.alert('No items found to push.');
    return;
  }

  const payload = {
    // clientName: clientName,
    lastUpdated: new Date().toISOString(),

  };

    'Push to Portal',
    'Found ' + items.length + ' item(s). Push to the live portal?',
    ui.ButtonSet.YES_NO

    pushToGitHub(jsonContent);
    ui.alert('Portal updated — ' + items.length + ' item(s) pushed.');

  }

function pushToGitHub(content) {
  const props = PropertiesService.getScriptProperties();
  const token = props.getProperty('GITHUB_TOKEN');
  const repo = props.getProperty('GITHUB_REPO');
  const path = props.getProperty('GITHUB_PATH') || 'data/client-data.json';
  const branch = 'main';  // ← Change if using a different branch

  if (!token || !repo) {
    throw new Error('Missing GITHUB_TOKEN or GITHUB_REPO in Script Properties.');
  }

  const apiUrl = 'https://api.github.com/repos/' + repo + '/contents/' + path;

  // Get current file SHA (needed for updates)
  let sha = '';
  try {
    const getResponse = UrlFetchApp.fetch(apiUrl + '?ref=' + branch, {
      method: 'GET',
      headers: {
        'Authorization': 'Bearer ' + token,
        'Accept': 'application/vnd.github.v3+json'
      },
      muteHttpExceptions: true
    });

    if (getResponse.getResponseCode() === 200) {
      sha = JSON.parse(getResponse.getContentText()).sha;
    }
  } catch (e) {
    // 404 = file doesn't exist yet, which is fine
  }

  // PUT the file
  const putPayload = {
    message: 'Update portal data — ' + new Date().toLocaleDateString('en-GB'),
    content: Utilities.base64Encode(content, Utilities.Charset.UTF_8),
    branch: branch
  };

  if (sha) putPayload.sha = sha;

  const putResponse = UrlFetchApp.fetch(apiUrl, {
    method: 'PUT',
    headers: {
      'Authorization': 'Bearer ' + token,
      'Accept': 'application/vnd.github.v3+json',
      'Content-Type': 'application/json'
    },
    payload: JSON.stringify(putPayload),
    muteHttpExceptions: true
  });

  const code = putResponse.getResponseCode();
  if (code !== 200 && code !== 201) {
    throw new Error('GitHub API returned ' + code + ': ' + putResponse.getContentText());
  }
}

function formatDateForJson(value) {
  if (!value) return "";
  if (value instanceof Date) {
    const y = value.getFullYear();
    const m = String(value.getMonth() + 1).padStart(2, '0');
    const d = String(value.getDate()).padStart(2, '0');
    return y + '-' + m + '-' + d;
  }
  const parsed = new Date(value);
  if (!isNaN(parsed.getTime())) {
    const y = parsed.getFullYear();
    const m = String(parsed.getMonth() + 1).padStart(2, '0');
    const d = String(parsed.getDate()).padStart(2, '0');
    return y + '-' + m + '-' + d;
  }
  return value.toString();
}
```
